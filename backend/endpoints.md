# Endpoints

## Auth

- POST /auth/signup
- POST /auth/signin
- POST /auth/signout
- GET /auth/me

## Pages

- POST /pages/query
- GET /pages/:id
- POST /pages
- POST /pages/:id/update
- POST /pages/:id/delete

## Comments

- POST /pages/:id/comments/query
- POST /pages/:id/comments
- POST /pages/:id/comments/:commentId/update
- POST /pages/:id/comments/:commentId/delete
